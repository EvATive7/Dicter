# Dicter
中文词库生成脚本框架，支持从多个给定的提供方中获取词汇。  
| 支持 | 描述 |  
| -- | -- |  
| ArknightsGameData | 从ArknightsGameData中提取适用于明日方舟的词库用词 |  
| CustomData | 从自定义文本文件合并词汇 |  
| 更多...? | 欢迎 PR |   

## 使用
1. `git clone`  
1. 新建配置保存到 `./data/config.yaml` （[示例](doc/config.yaml)）
1. 满足Providers的前提
1. `python main.py`
